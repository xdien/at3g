at3g
====

chương trình hỗ trợ chuyển đổi 2g/3g(edge/umts) một số thiết bị USB 3g trên linux 

-Thiết bị được hỗ trợ
	
	+sierra wireless 885
	-E173u-1 thất bại(tự ngắt kết nối)
-Cài đặt
*yêu cầu phải có plugins qtserialport cho QT

	qmake .
	make
	
-Chạy trương trình

	./at3g

-Đang làm

+Hỗ trợ thêm mf190
